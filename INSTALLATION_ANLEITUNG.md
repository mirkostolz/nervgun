# Nervgun Extension - Installationsanleitung

## Für Entwickler (Mirko)
### Extension für Kollegen vorbereiten:

1. **ZIP-Datei erstellen:**
   ```bash
   cd /Users/mirkostolz/Desktop/Code/Nervgun
   zip -r nervgun-extension.zip extension/
   ```

2. **ZIP-Datei teilen:**
   - Per E-Mail anhängen
   - Oder in Cloud-Speicher (Google Drive, Dropbox) hochladen
   - Link an Kollegen senden

## Für Kollegen
### Extension installieren:

1. **ZIP-Datei herunterladen** (von Mirko erhalten)

2. **Chrome öffnen** und zu `chrome://extensions/` navigieren
   - Oder: Chrome Menü → Weitere Tools → Erweiterungen

3. **Entwicklermodus aktivieren:**
   - Oben rechts den Schalter "Entwicklermodus" einschalten

4. **Extension laden:**
   - "Entpackte Erweiterung laden" klicken
   - Den `extension` Ordner aus der ZIP-Datei auswählen
   - Oder: ZIP-Datei direkt auswählen (Chrome entpackt automatisch)

5. **Fertig!** 
   - Die Extension erscheint in der Liste
   - Extension-Icon sollte in der Chrome-Toolbar sichtbar sein

## Extension verwenden:

- **Klick auf Icon:** Öffnet das nervgun Popup
- **Tastenkürzel:** `Ctrl+Shift+A` (Windows) oder `Cmd+Shift+A` (Mac)
- **Funktion:** Schnelle Erfassung von Ärgernissen

## Bei Problemen:

1. **Extension nicht sichtbar?**
   - Chrome neu starten
   - Entwicklermodus prüfen

2. **Fehler beim Laden?**
   - ZIP-Datei nochmal herunterladen
   - Chrome auf neueste Version aktualisieren

3. **Fragen?**
   - Mirko kontaktieren

## Updates erhalten:

Wenn Mirko eine neue Version erstellt:
1. Neue ZIP-Datei herunterladen
2. Alte Extension in Chrome deaktivieren/löschen
3. Neue Version installieren (siehe Schritte oben)
